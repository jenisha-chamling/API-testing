# API Testing Project

## Project Overview

This project demonstrates API testing of a custom REST API developed using JSON Server. The API was created locally to simulate real-world backend functionality and supports complete CRUD (Create, Read, Update, Delete) operations.

Testing was performed using Postman, while automated test execution and HTML report generation were carried out using Newman CLI.

## Features

* Custom REST API development using JSON Server
* CRUD operation testing:

  * GET – Retrieve data
  * POST – Create new data
  * PUT/PATCH – Update existing data
  * DELETE – Remove data
* HTTP status code validation
* Response body verification
* Negative test scenario validation
* Automated test execution using Newman
* HTML report generation for test results
