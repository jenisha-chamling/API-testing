This project demonstrates API testing of a custom REST API created using JSON Server. The API was built locally to simulate real-world backend behavior and supports full CRUD operations (Create, Read, Update, Delete). The API was tested using Postman, and automated test execution and reporting were done using Newman CLI.

 Features:
Custom REST API development using JSON Server
CRUD operations:
GET → Fetch data
POST → Add data
PUT/PATCH → Update data
DELETE → Remove data
HTTP status code validation
Response body verification
Negative testing scenarios
Automated test execution using Newman
HTML report generation
